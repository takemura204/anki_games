## コアコンセプト

**「ゲームが学習を引き出し、学習がゲームを強化する」双方向ループ。**

- **ゲームが学習を引き出す**: SRSが毎日の起動動機になる（今日復習できる単語が積み重なる）
- **学習がゲームを強化する**: 正解するほど有利な状況が生まれる → スコアにも直結 → 学習が報われる

パズルのみのゲーム（1010! / 2048等）に比べ継続率が高く、Duolingoのような学習専用アプリに比べ「ゲームを楽しむついでに覚える」低ハードルな導入が可能。

## ゲームタイプ

アプリは **2種類のゲーム** を提供する。どちらも英単語クイズと連動する。

### Block Puzzle（ブロックパズル）
8×8グリッドにブロックを配置し、ラインを消去するパズル。正解数がピース品質（Easy/Medium/Hard）に影響する。セーブ・ロード対応。

### Number Merge（ナンバーマージ）
4×4盤面で同じ数字タイルをスワイプして合体させる2048系パズル。英単語クイズと連動し、正解すると有利なタイルがスポーンされる。

```
HomeScreen:
  ┌─────────────────────────────────┐
  │  ⚙️（設定ボタン）                │
  │                                 │
  │  [ロゴ]                          │  ← PageView（左右スワイプで切り替え）
  │  BLOCK PUZZLE / NUMBER MERGE    │
  │  ● ○（ページドット）             │
  │                                 │
  │  [英検5] [英検4] [英検3] [準2]  │
  │  [英検2] [600]  [700]  [800]   │  ← レベルカードグリッド（4列）
```

- 上部 PageView でゲームタイプ（Block Puzzle / Number Merge）を切り替え

- レベルカードをタップ → **QuizStartSheet** が開く

## 英単語クイズ詳細

### ゲームフロー

```
HomeScreen（レベルカード選択）
        ↓
QuizStartSheet（習熟度フィルター・順序選択）
        ↓ START
ブロック/マージフェーズ → クイズフェーズ（3問）→ 繰り返し
        ↓
ゲームオーバー（セッション統計・苦手単語リスト）
```

### クイズメカニクス

- **1ラウンド3問**
- **TTS**: 英単語表示時に自動読み上げ + タップで再生（`flutter_tts`）
- **ステージ別出題形式**:

| ステージ | 形式 | 内容 |
|---------|------|------|
| 0〜2 | スワイプ4択（英→日） | カードをスワイプして日本語訳を選ぶ |
| 3 | letterTap | 日本語を見てスペルを文字タップで入力 |
| 4 | タイピング | 日本語を見て英単語をキーボード入力 |

### SRS（間隔反復）アルゴリズム

単語ごとにステージ（0〜5）とインターバルを管理し、科学的な復習タイミングで出題する。

| ステージ | 表示グループ | 基準インターバル |
|---------|------------|---------------|
| 0 | New（未学習） | 即出題 |
| 1 | Hard（苦手） | 4時間 |
| 2 | Learning（学習中） | 24時間 |
| 3 | Good（得意） | 72時間 |
| 4 | Perfect（完璧） | 168時間（1週間） |
| 5 | Perfect（完璧） | 720時間（1ヶ月） |

- 正解でステージ昇格（累計正解数に応じて）
- 不正解で weight ≥ 3.5 になるとステージ降格
- 期限超過単語を正解すると超過日数ボーナス（+10〜100pt）

### 出題順序モード

| モード | 内容 |
|--------|------|
| おまかせ（auto） | SRS・重み付きランダム |
| 苦手優先（masteryLowToHigh） | ステージ昇順・同ステージ内は重み降順 |
| 得意優先（masteryHighToLow） | ステージ降順・同ステージ内は重み昇順 |
