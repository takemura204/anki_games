## コアコンセプト

Block. は **8×8グリッドにブロックを配置して行・列を消すパズルゲーム**。
複数のゲームモードを通じて異なる体験を提供する。
英単語モード（Quiz Mode）では **「ブロックパズルをご褒美に、英単語学習を続けさせる」** というコンセプトで、ゲームループと語学学習を統合する。

---

## ゲームモード

### 1. クラシックモード（CLASSIC）

- **概要**: 制限なしのエンドレスパズル
- **目的**: スコアを最大化する
- **終了条件**: 盤面にピースを置けなくなったらゲームオーバー
- **継続機能**: リワード広告視聴で1回だけコンティニュー可能
- **スコア保存**: ハイスコアをローカルに永続保存
- **セーブ機能**: 中断データを保存して再開可能

### 2. クエストモード（QUEST）

- **概要**: レベルクリア型のパズル
- **目的**: 目標スコアを達成してレベルをクリアする
- **仕様**:
  - 各レベルに目標スコアが設定される
  - クリアすると次のレベルが解放される（最大解放レベルを永続保存）
  - 失敗すると「LEVEL FAILED」オーバーレイが表示される
  - 「Retry Level」でそのレベルを再挑戦可能
- **セーブ機能**: 中断データを保存して再開可能

### 3. タイムアタックモード（TIME ATTACK）

- **概要**: 90秒間でスコアを競う
- **仕様**:
  - 画面上部にタイマーバーを表示
  - カウントダウン演出あり（3秒）
  - 時間切れで結果オーバーレイを表示
  - ハイスコアをローカルに永続保存

### 4. 英単語モード（WORD MODE / Quiz Mode）

- **概要**: 英単語スワイプクイズ＋ブロックパズルの融合モード
- **コンセプト**: ブロック配置をご褒美として英単語学習を促進する

#### ゲームフロー

```
学習範囲選択画面
        ↓ START
   ブロック配置フェーズ
        ↓（ターン終了 or 一定条件）
   クイズフェーズ（3問）
     ・スワイプで4択回答
     ・3問答えたら結果表示
        ↓
   ピース付与（正誤・充填率で難易度決定）
        ↓
   ブロック配置フェーズ（繰り返し）
        ↓
   ゲームオーバー（正解率・苦手単語リスト表示）
```

#### クイズ設問

- 1ラウンドで3問出題
- 各問題はカードをスワイプして回答（↑↓←→の4択）
- 出題方向: 英→日 / 日→英 / ランダム（設定で変更可）
- デコイ: 同カテゴリの単語を優先的に選出

#### ピース難易度割り当て（充填率×正誤の2軸）

| 条件 | ピース難易度 |
|------|------------|
| 充填率 ≥ 75% | 常にEasy（小ピース） |
| 充填率 ≥ 65% かつ不正解 | Medium（中ピース）|
| 正解 | Easy |
| 不正解 | Hard（大ピース）|

- Easy: 1〜3セル
- Medium: 4セル
- Hard: 5セル以上

#### スコア・倍率設計

- `quizMultiplier`: ライン消去ごとに+1、クイズ全問正解でも+1（リセット: 不正解あり）
- `combo`: 連続ライン消去カウント

#### 学習アルゴリズム（SRS: Spaced Repetition System）

| 操作 | weight の変化 |
|------|-------------|
| 初回 | 1.0（DB未登録）|
| 正解 | × 0.7（最小0.1）|
| 不正解 | × 1.5（最大5.0）|

習熟度4段階:

| 段階 | 条件 | 色 |
|------|------|----|
| 未学習 | DB未登録 | グレー |
| 苦手 | weight ≥ 2.0 | 赤 |
| 得意 | 0.5 ≤ weight < 2.0 | 青 |
| 完璧 | weight < 0.5 | 緑 |

#### 不正解単語の優先出題

- セッション内で不正解になった単語は weight を一時的に2倍にして次ラウンドで優先出題される
- この一時増幅はDBには保存されず、ゲーム再起動でリセット

#### 学習範囲フィルター

習熟度系:
- すべての単語
- 苦手＋未学習
- 苦手のみ

資格レベル系:
- 英検5級
- 英検4級
- 英検3級
- TOEIC基礎

## テーマ・スキン

ゲームテーマはアプリ内で変更可能。各テーマは**セルのレンダリングスタイル・背景・消去エフェクト・効果音**のセットを持つ。

### セルレンダリングモード（CellRenderMode）

| スタイル | 特徴 |
|---------|------|
| glassmorphism | 半透明＋縁グロー |
| wireframe | ボーダーのみ発光 |
| glossy | ツヤ＋大角丸＋ハイライトバブル |
| matte | 低彩度＋微細ノイズ |
| bubble | ビニール、強いハイライト |
| ice | 氷、内部クラック |
| slime | 粘液、有機的表面 |
| slate | 純黒縦グラデーション |

### 消去エフェクト（ClearEffectMode）

flash / glitch / confetti / ripple / dice / dissolve / pop / shatter

### 背景モード（BackgroundMode）

solid / cyberGrid / gradient / paperTexture
