## コアコンセプト

**「ゲームが学習を引き出し、学習がゲームを強化する」双方向ループ。**

- **ゲームが学習を引き出す**: SRSが毎日の起動動機になる（"今日N語レビューできます"）
- **学習がゲームを強化する**: 正解するほど有利なピースが来る → スコアにも直結 → 学習が報われる

パズルのみのゲーム（1010!等）に比べ継続率が高く、Duolingoのような学習専用アプリに比べ「ゲームを楽しむついでに覚える」低ハードルな導入が可能。

## ゲームモード

### クラシックモード
エンドレスパズル。ハイスコア更新が目標。リワード広告でコンティニュー可能。セーブ・ロード対応。

### クエストモード
目標スコアを達成してレベルを解放するステージ制パズル。最大解放レベルを永続保存。

### タイムアタックモード
90秒間のスコアアタック。タイマーバー表示・カウントダウン演出あり。ハイスコア保存。

### 英単語モード（Word Mode）
**本アプリの核心機能。** ブロックパズル×英単語スワイプクイズの融合。

## 英単語モード 詳細

### フロー

```
HomeScreen（資格レベル選択）
        ↓
学習ハブ（円グラフ・ランキング・フィルター）
        ↓ START
ブロック配置フェーズ → クイズフェーズ（3問）→ ピース付与 → 繰り返し
        ↓
ゲームオーバー（正解率・苦手単語リスト）
```

### クイズメカニクス

- **出題**: 1ラウンド3問、カードをスワイプ（4択）で回答
- **TTS**: 英単語表示時に自動読み上げ + タップで再生（`flutter_tts`）
- **出題方向**: 英→日 / 日→英 / ランダム（学習ハブで選択）
- **ピース難易度**: 充填率とストリーク正解数に応じてEasy/Medium/Hardが変わる

### SRS（間隔反復）アルゴリズム

単語ごとにステージ（0〜5）とインターバルを管理し、科学的な復習タイミングで出題する。

| ステージ | 意味 | 基準インターバル |
|---------|------|---------------|
| 0 | 未学習 | 即出題 |
| 1 | 苦手 | 4時間 |
| 2 | 学習中 | 24時間 |
| 3 | 得意 | 72時間 |
| 4 | 上級 | 168時間（1週間） |
| 5 | 完璧 | 720時間（1ヶ月） |

- 正解でステージ昇格（累計正解数: 2→3→4→5→6回ごと）
- 不正解で weight ≥ 3.5 になるとステージ降格
- 期限超過単語を正解すると超過日数ボーナス（+10〜100pt）

### 学習ハブ機能

- **習熟度donutチャート**: 未学習/苦手/得意/完璧の分布を円グラフで表示
- **得意・苦手TOP5ランキング**: 2カラム表示
- **フィルター**: 習熟度（全て/苦手+未学習/苦手のみ）×資格レベル（英検5/4/3/TOEIC）のAND条件
- **ジャンルフィルター**: 頻出50選 + nature/daily/people等7ジャンル（プレミアム限定）
- **単語一覧ボトムシート**: 検索・ステータスフィルター・TTS再生・手動ステータス変更

### 単語データ

| レベル | 語数 |
|--------|------|
| 英検5級 | 約100語（頻出50語マーク付き） |
| 英検4級 | 約100語 |
| 英検3級 | 約150語 |
| 英検2級 | 約100語 |
| TOEIC基礎 | 約100語 |
| **合計** | **500語以上** |

スキーマ: `id, en, ja, category, level, is_frequent, example_en, example_ja`

## マネタイズ

### モデル: 広告＋サブスクのフリーミアム

| 機能 | 無料 | プレミアム（¥480/月） |
|------|------|-------------------|
| バナー広告 | 常時表示 | 非表示 |

- **インタースティシャル**: ゲームオーバー時（クラシック/クエスト）
- **リワード**: クラシックモードのコンティニュー
