# WordMode 画面リデザイン 仕様書

> 作成日: 2026-03-22 / ステータス: Approved

## 概要

WordMode（英単語×パズル）の選択画面を全面リデザインする。
資格レベル選択を HomeScreen へ移し、WordMode 画面をダッシュボード・ランキング・単語一覧を統合したハブ画面に刷新する。

## 要件

### 機能要件

1. **HomeScreen**: WordMode カード内に資格レベルチップ（英検5/4/3/TOEIC）を常時表示。タップで即時フィルター切替。
2. **WordMode 画面（1画面スクロール）**
   - 円グラフダッシュボード（左: donutチャート + 中央%、右: 4ステータス内訳）
   - 学習範囲フィルター（横3チップ択一）
   - 得意TOP5 / 苦手TOP5 ランキング（2カラム）
   - スタートボタン（件数表示なし）
   - ヘッダー右端の ⋮ ボタンで単語一覧ボトムシートを開く
3. **単語一覧ボトムシート**
   - 検索バー（英/日リアルタイム絞り込み）
   - ステータス別フィルタータブ（全て/未学習/苦手/得意/完璧）
   - 各行: 英語 | 日本語 | 🔊 | ステータスバッジ（タップで変更メニュー）
4. **ViewModel 追加機能**
   - `WordEntry` モデル（word + stage + weight）
   - `wordEntries`（全単語+ステージ）をステートに保持
   - `topBestWords` / `topWorstWords` TOP5 リスト
   - `updateWordStage(wordId, stage)` 手動ステータス変更

### 非機能要件

- 外部チャートパッケージ不使用（CustomPainter で実装）
- very_good_analysis lint 準拠
- 既存クイズゲームロジックへの影響なし

## 実装方針

| ファイル | 変更内容 |
|---------|---------|
| `quiz_view_model.dart` | `WordEntry` 追加、ステート拡張、`updateWordStage` 追加 |
| `word_range_selector_screen.dart` | 全面リライト |
| `home_screen.dart` | WordMode カードに level chips 追加 |
| `mode_card.dart` | `bottomContent` パラメータ追加 |
| `ja.i18n.json` / `en.i18n.json` | 新規キー追加 |

### ランキング計算ロジック
- **得意TOP5**: `stage >= 3`、stage 降順、上位5件
- **苦手TOP5**: `stage == 1`、weight 降順（出題重み高 = 苦手度高）、上位5件
- 選択レベルフィルターで絞り込み後に計算
